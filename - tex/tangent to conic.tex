A general equation for any conic section is
\[ Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0 \]

If $(x_0,y_0)$ lies on the conic section, then it satisfies the above equation.  Now, he says, let
\[ Ax^2 + Bxy + Cy^2 + Dx + Ey + F = \]
\[ \ \ \ \ \ \ \ \  \ \ \ \ \ \ \ \ A(x-x_0)^2 + B(x-x_0)(y - y_0) + C(y-y_0)^2 \]

If the right-hand side is not zero, then this is the equation of a line passing through $(x_0,y_0)$ where no other point on the line is on the conic section.  In other words, it is the equation of the tangent line.

The example given is
\[ x^2 - xy + y^2 - 7 = 0 \]

The point $(-1,2)$ is on the curve.

Plugging into what we had above, the tangent line through $(-1,2)$  is
\[ x^2 - xy + y^2 - 7 = (x+1)^2 - (x+1)(y-2) + (y-2)^2 \]
\[ -7 = 2x + 1 +2 x - y + 2 - 4y + 4 \]
\[ 5y = 4x + 14 \]
\[ y = \frac{4}{5} x + \frac{14}{5} \]

A second example is the unit circle.  We have
\[ x^2 + y^2 - 1 = 0 = (x-x_0)^2 + (y - y_0)^2 \]
\[ -1 = -2xx_0 + x_0^2 - 2yy_0 + y_0^2 \]
\[ 2yy_0 = -2xx_0 + x_0^2 + y_0^2 + 1 \]
\[ y = - \frac{x_0}{y_0} \ x + \ [ \ \frac{x_0^2}{2y_0} + \frac{y_0}{2} + 1 \ ] \]
The \emph{slope} of the tangent to a unit circle at any point $(x,y)$ is $-x/y$.

The easiest calculus derivation uses implicit differentiation:
\[ 2x \ dx + 2y dy = 0 \]
\[ \frac{dy}{dx} = - \frac{x}{y} \]
\checkmark

For the general parabola in standard orientation:
\[ ax^2 + bx + c - y = 0 \]
write
\[ ax^2 + bx + c - y = a(x-x_0)^2 + b(x-x_0) - (y - y_0) \]
And we have a problem in this case, because the $y$ term disappears.
\[ c = - 2ax_0 x + ax_0^2 - bx_0 - y_0 \]
If we did have a $+y$ on the right-hand side then the slope would be correct, since the minus sign would go leaving $2ax_0 x$ on the left-hand side.

So if we retreat to the equation for a parabola in any orientation, the equation from above is:
\[ Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0 \]
which is a parabola \emph{if} $B^2 = 4AC$.  So for example, 
\[ x^2 + 2xy + y^2 + x - y + 4 = 0 \]
is a parabola, tilted to the left with respect to the $y$-axis.

If we apply the method described in the link:
\[ x^2 + 2xy + y^2 + x - y + 4 = (x-x_0)^2 + 2(x-x_0)(y - y_0) + (y-y_0)^2 + x - x_0 - y + y_0 \]
$x$ and $y$ cancel
\[ x^2 + 2xy + y^2 + 4 = (x-x_0)^2 + 2(x-x_0)(y - y_0) + (y-y_0)^2 - x_0 + y_0 \]

Next multiply out and cancel
\[ 4 =  -2x x_0 + x_0^2 - 2x_0y  - 2xy_0 + 2x_0 y_0 - 2yy_0 + y_0^2 - x_0 + y_0 \]
Group the terms with $y$ on the left and $x$ on the right:
\[ \ [ \ 2x_0 + 2y_0 \ ] \ y =  \ [ \ -2 x_0 - 2y_0 \ ] \ x + \ [ \ x_0^2  + 2x_0 y_0 + y_0^2 - x_0 + y_0 - 4 \ ] \]
\[ \ [ \ 2x_0 + 2y_0 \ ] \ y =  \ [ \ -2 x_0 - 2y_0 \ ] \ x + \ [ \ (x_0 + y_0)^2 \ ] \ - \ [ \ x_0 + y_0 \ ]  - 4 \]

Except for that trailing $4$ we can get lots of cancelations.  However, what this says is that
\[ y = -x + k \]
which is clearly wrong.  The slope of the tangent is not $-1$ everywhere.
